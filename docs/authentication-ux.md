# Authentication UX Improvements

Branch: `feature/user-authentication`  
Related Issue: #1

## Overview
This branch focuses on usability and accessibility improvements
for the login interface rather than backend authentication logic.

## Features Added

### Remember Email
- Allows users to save their email address locally
- Uses browser `localStorage`
- Improves convenience for frequent users

### Caps Lock Warning
- Displays a warning when Caps Lock is enabled
- Helps prevent password entry mistakes
- Improves accessibility and user awareness

## Notes
All features are front-end only and intended for demonstration
purposes. Backend authentication will be implemented separately.
